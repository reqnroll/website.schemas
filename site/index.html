Reqnroll JSON Schemas

This website is a collection of JSON schema files to be used with Reqnroll.

Reqnroll is an open-source .NET framework for Behavior Driven Development (BDD) that allows you to write executable specifications in plain language.

Please check our GitHub repository for details, licenses and open-source contribution guides: https://github.com/reqnroll

Available Schemas

  • reqnroll-config-latest.json - Configuration schema for the latest version of Reqnroll
  • reqnroll-config-3.2.json - Configuration schema for Reqnroll 3.2
  • reqnroll-config-3.1.json - Configuration schema for Reqnroll 3.1
  • reqnroll-config-3.0.json - Configuration schema for Reqnroll 3.0
  • reqnroll-config-1.1.json - Configuration schema for Reqnroll 1.1
  • reqnroll-config-1.0.json - Configuration schema for Reqnroll 1.0
